     70 ### Working
     71 
     72 To help on my working time, I use these apps:
     73 
     74 - **Ripcord** chat client for Slack and Discord
     75   <small>The official **Slack** and **Discord** desktop client uses **Electron**, I don't like that.
     76   [Ripcord][ripcord] is a shareware that is based on, I don't know, it looks like **GTK+**.</small>
     77 - **Xit** Git GUI for macOS
     78   <small>When doing heavy refactor, I like to code while looking at the repository's history.
     79   [Xit][xit] is a free and open source. I think it is comparable to the awesome **Sublime Merge**.</small>

    101 ### Previous Apps
    102 
    103 - **Canary mail** → Apple Mail
    104   <small>lightweight email client with configurable keyboard shortcuts, and PGP encryption</small>
    105 - **Reeder 4** → **NetNewsWire**
    106   <small>beautiful and gesture-based RSS Reader.</small>
    107 - **BusyCal** → Apple Calendar
    108 - **Emacs** → **Obsidian**
    109   <small>Org-roam is an awesome minor package for Org-mode.
    110   Read [my post about this](/posts/2020-06-org-mode-with-org-roam/).</small>
    111 - **Worldbrain's Memex 2** → **DEVONthink**
    112   <small>Browser extension to bookmark, highlight and annotate any site.
    113   With full text search and grouping, this is a good service for knowledge gathering.</small>
    114 - **Historio.us** → **DEVONthink**
    115   <small>Web archiver. It supports full text search and tagging.
    116   I use this side-by-side with **Memex 2**.</small>
    117 - **aText** → **Keyboard Maestro**
    118   <small>alternative to TextExpander that is single purchase.
    119   my template to make Zettelkasten [is here](/blogging-and-noting.atext).</small>
