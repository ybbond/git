     64 ### Working
     65 
     66 To help on my working time, I use these apps:
     67 
     68 - **Xit** Git GUI for macOS
     69   <small>When doing heavy refactor, I like to code while looking at the repository's history.
     70   [Xit][xit] is a free and open source. I think it is comparable to the awesome **Sublime Merge**.</small>
     71 - **Ripcord** chat client for Slack and Discord
     72   <small>The official **Slack** and **Discord** desktop client uses **Electron**, I don't like that.
     73   [Ripcord][ripcord] is a shareware that is based on, I don't know, it looks like **GTK+**.</small>

     75 ### Services
     76 
     77 - **Fastmail**
     78   <small>I try to use less Google, but I cannot find replacement for ~~YouTube~~ and Translate :(</small>
     79   <small>Thanks to [Josh Habdas](https://habd.as), I now use Invidio.us</small>
     80 - **BitWarden** password manager
     81 - **Worldbrain's Memex 2**
     82   <small>Browser extension to bookmark, highlight and annotate any site.
     83   With full text search and grouping, this is a good service for knowledge gathering.</small>
     84 - **Historio.us**
     85   <small>Web archiver. It supports full text search and tagging.
     86   I use this side-by-side with **Memex 2**.</small>
